# Пост с картинкой и текстом (текст под картинкой)

1. При создании выбираем тип поста «Фото»:

   ![image](/images/105613117-94be9c80-5dd1-11eb-89ee-dcdacbd95d1c.png)

2. Нажимаем кнопку <kbd>Выбрать файл</kbd> и загружаем картинку:

   ![image](/images/105613135-b28c0180-5dd1-11eb-85a0-d944a5444092.png)

3. После загрузки картинки можно добавить текст (не более 1024 символов):

   ![image](/images/105613150-e0714600-5dd1-11eb-82d0-66d50b073f8b.png)

После этого пост будет состоять из картинки, под которой будет текст. Пример:

![image](/images/105613165-026ac880-5dd2-11eb-97f1-7feff0fc0d27.png)
