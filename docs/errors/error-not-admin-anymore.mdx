# Ошибка `NOT_ADMIN_ANYMORE`

В случае, если пользователь подключает каналу к Телепосту через бота [{{ telepost.bot.name }}]({{ telepost.bot.url }}), и после этот пользователь выходит из канала / перестает быть в нем админом, то при попытке опубликовать пост может появиться ошибка `NOT_ADMIN_ANYMORE`.

## Решение

Пользователю, подключившему канал через бота [{{ telepost.bot.name }}]({{ telepost.bot.url }}) — нужно снова стать админом в канале. Также необходимо передобавить канал в Телепост: не удалять канал из Телепоста (иначе пропадет история постов в Телепосте), а просто добавить заново.

Также рекомендуется использовать свой кастомный бот, как его подключить — читайте в [этой инструкции](../start/personal-bot-for-telepost.mdx).
