# Ошибка `BOT_ACCESS_FORBIDDEN`

Иногда может появиться сообщение "Channel deactivated" и ошибка `BOT_ACCESS_FORBIDDEN`:

![image](/images/202869020-37e868be-5841-44ca-984d-cb26db11add4.jpg)

![image](/images/202869049-0eeb62d0-6ec3-439a-9ca4-ef8a9b0cd2a4.jpg)

В таких случаях помогает переключить подключенных ботов в веб-интерфейсе: переключить на системного бота [{{ telepost.bot.name }}]({{ telepost.bot.url }}), а потом вернуть обратно кастомного бота (или наоборот, если у вас по умолчанию используется системный бот).

Рекомендуется использовать свой кастомный бот, как его подключить — читайте в [этой инструкции](../start/personal-bot-for-telepost.mdx).
